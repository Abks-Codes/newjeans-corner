# NewJeans Corner 🐇

A fan-made NewJeans website celebrating the members, music videos, and socials of NewJeans. The site includes an interactive homepage where users can click each member to learn more about them, plus a songs page with an embedded music video player.

## ✨ Features

* Interactive member cards
* Member information updates when clicked
* Background color changes based on the selected member
* Emoji changes for each member
* Embedded YouTube music video player
* Clickable song list
* Responsive layout for mobile and desktop
* Smooth animations and hover effects

## 🛠️ Built With

* HTML
* CSS
* JavaScript
* Google Fonts
* YouTube Embed

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.

Or, if it is published with GitHub Pages, visit the live demo link above.

## 📸 Pages

### Home Page

The home page displays NewJeans member cards. When a user clicks a member, the page updates with their name, age, birthday, fun fact, nickname, emoji, and background color.

### Songs Page

The songs page includes an embedded YouTube music video player. Users can click on different song titles to change the video.
